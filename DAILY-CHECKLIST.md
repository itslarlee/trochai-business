## MORNING BLOCK (30-45 min) — Loom recording

- [ ] Open today's Loom script from `loom-scripts/`
- [ ] Pre-flight: open the prospect's website + LinkedIn, pick ONE specific property/detail
- [ ] Open demo WhatsApp `+506 6404 2819` and demo inbox `trochai-real-estate.trochai.com`
- [ ] Record Loom (3-4 min max) — rerecord if over 4:30
- [ ] Send Loom + one-line WhatsApp message (template in each script file)
- [ ] Update `leads/master-leads.csv` → Status column: **"Loom Sent YYYY-MM-DD"**

**Target: 2-3 Looms before lunch.** If you can't do 2, do 1 — don't skip a day.

---

## MIDDAY BLOCK (30 min) — LinkedIn warming

- [ ] Check LinkedIn notifications — who accepted connections?
- [ ] Like **5-10 posts** from ICP prospects (real estate agencies in CR)
- [ ] Leave **1-2 thoughtful comments** on posts from Tier A prospects (3+ sentences, add value, don't pitch)
- [ ] Send **5-10 new connection requests** — only to people whose post you commented on in the last 48h (warm-before-connect)
- [ ] For anyone who's accepted and is on the `master-leads.csv` P1/P2 list: note it, they're priority for a Loom later

**DO NOT pitch in LinkedIn DMs.** The DM is for "thanks for connecting", nothing else. Looms go via WhatsApp.

---

## AFTERNOON BLOCK (30-45 min) — More Looms + replies

- [ ] Check WhatsApp for replies to previous Looms
- [ ] For any "sí" / "me interesa" reply → send scheduling link or propose 2-3 time slots
- [ ] For "no gracias" / "no me interesa" → thank them, mark CSV **"No Interest"**, close the loop
- [ ] Record 1-2 more Looms for next P1 leads (target: **3-5 total Looms/day**)
- [ ] Reply to any LinkedIn DMs

## FRIDAY BLOCK — Mystery shop batch (2 hrs, replaces afternoon block on Fridays)

**Goal:** Generate real response-time data for next week's Looms.

- [ ] Pick **5-10 P1/P2 leads** you plan to pitch next week
- [ ] For each: find the **AGENCY's public WhatsApp** on their website (NOT the CSV number, which may be direct)
- [ ] Send the buyer message from `loom-scripts/MYSTERY-SHOP-TEMPLATE.md` matching their zone
- [ ] Log send timestamp in a spreadsheet (Agency / Sent at / Reply at / Delta / Quality)
- [ ] **Do not reply yet** — wait for Monday

## MONDAY MORNING — Close the mystery shop loop

- [ ] For every agency that replied: send the closing message *"Perfecto, gracias. Sigo viendo opciones, le escribo si algo me calza."*
- [ ] For every agency that didn't reply within 72h: log "No reply — 72h+" — that's your data
- [ ] Feed the timing data into this week's Loom scripts (use `02-alejandro-gonzalez-casamax.md` as the template)

## WHEN YOU GET STUCK

| Situation | Action |
|---|---|
| Sent 10 Looms, 0 replies | Stop. Come back and redo the script with Claude — one of: hook, CTA, or targeting is broken |
| Got a reply "me interesa" | Reply in <1 hour. Propose 2-3 times in next 48h. Use Calendly if you have it |
| Got into a demo call | Follow the playbook in `LOOM-OUTREACH-SYSTEM.md` §5 (closing without sales calls) |
| Ran out of P1 leads | Move to P2 (LinkedIn-only contacts). Warm via comment → connect → DM → ask for WhatsApp |
| Demo account broke | Fix it. The demo is the entire product of the Loom. Broken demo = zero sales |
