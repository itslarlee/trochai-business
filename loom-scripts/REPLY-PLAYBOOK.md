# Mystery Shop — Reply Playbook

**Keep this open during the shop.** When an agency replies, look up the scenario and copy-paste the response.

## Golden rules

1. **Never lie beyond "looking at options."** You're a buyer in exploration phase — that's your whole role. No fake IDs, no fake employer stories, no invented family situations.
2. **Never schedule a visit or call.** Close the loop politely if asked.
3. **Never commit to anything** — "te compro hoy", "deposito mañana", etc.
4. **Reply within 10 min** if you can — don't ghost people who replied to your mystery shop.
5. **After 1-2 exchanges, close the conversation.** Use the CLOSE-LOOP message below.
6. **Don't give your real last name or email.** If they ask, deflect with "le dejo el WhatsApp como principal."

## 🎯 Scenario templates (copy-paste based on their reply)

### 1. Instant autoreply (no human yet)

**Just log the timestamp.** Don't reply to autoreply — wait for the real human. Note in tracker: *"Autoreply received at HH:MM"*.

Example autoreply you might get: *"Gracias por comunicarse, le responderemos en breve."*

---

### 2. Generic "¡Hola! ¿En qué te puedo ayudar?" (no listings, no qualifying)

They're just acknowledging. Give them ONE more piece of info, then close.

**ES:**
```
Gracias. Vivo fuera de Costa Rica pero estoy evaluando opciones para vacacionar / mudarme en los próximos meses. Si tienen algo que calce con el perfil, excelente. Si no, les agradezco el tiempo.
```

**EN:**
```
Thanks. I'm based outside Costa Rica but exploring options for the coming months. If you have anything matching that profile, great. If not, I appreciate the time.
```

Then wait for their next move. If they send listings → scenario 4. If they ask more → scenario 3. If they don't reply → done.

---

### 3. They ask qualifying questions ("¿zona específica?", "¿para vivir o inversión?", "¿cuándo compraría?")

Answer briefly in buyer-tone, don't commit to a hard timeline.

**Zone question (ES):**
```
Me interesa más [Escazú / Santa Ana / Tamarindo — repeat the zone from your original message]. No estoy cerrado a otras opciones si el precio/ubicación calza.
```

**Zone question (EN):**
```
I'm most interested in [zone]. Not closed to other options if price + location works.
```

**Purpose question (ES):**
```
Mayormente para vivir, aunque no descarto renta si la propiedad lo permite.
```

**Purpose (EN):**
```
Mostly to live in, though I'd consider short-term rental if the property fits.
```

**Timeline question (ES):**
```
No tengo prisa — estoy en fase de información, probablemente 3-6 meses. Primero quiero ver qué hay en el mercado.
```

**Timeline (EN):**
```
No rush — I'm in research phase, probably 3-6 months out. First want to see what's available.
```

**Financing question (ES):**
```
Cash o una combinación — aún evaluando. ¿Tienen financiamiento disponible?
```
*(Flip the question back — it's realistic buyer behavior)*

---

### 4. They send listings / photos / links

**Perfect — you got your data point (they replied + quality = "sent listings").**

**ES:**
```
Excelente, gracias por mandarme las opciones. Les voy a echar un ojo con calma y les escribo si algo me calza. Agradecido por el tiempo.
```

**EN:**
```
Great, thanks for sending those. I'll review them carefully and reach out if something fits. Appreciate the time.
```

Don't engage further. Close logged.

---

### 5. They ask to schedule a call / video / visit

**Deflect politely. You can't take a real call.**

**ES:**
```
Agradezco la oferta, pero por ahora prefiero seguir por mensaje — todavía estoy ordenando ideas. Si algo de lo que me recomienden me llama la atención, coordinamos con más tiempo.
```

**EN:**
```
Appreciate the offer, but for now I'd rather keep it in writing — still organizing my thoughts. If something catches my eye from what you recommend, we can coordinate a call later.
```

If they push: *"De verdad, por ahora solo estoy viendo opciones, gracias."*

---

### 6. They ask for your full name / email / passport / ID

**Deflect. Don't invent fake ID info — that crosses into outright deception.**

**ES:**
```
Prefiero dejar el WhatsApp como contacto principal por ahora — cuando algo me llame la atención le paso más detalles.
```

**EN:**
```
I'd rather keep WhatsApp as the main contact for now — I'll share more details once something grabs my interest.
```

If they insist they need your name for listings, just use your first name only: *"Soy Leeren"* (or whatever first name you want). No lying required.

---

### 7. They ask what city / country you're in (qualifying the international angle)

**Keep it vague but plausible.**

**ES:**
```
Estoy en [Miami / Canadá / España — pick one realistic city]. He ido a Costa Rica varias veces y ahora estoy pensando serio en una segunda casa / mudanza.
```

**EN:**
```
Based in [Miami / Toronto / NYC]. Been to Costa Rica a few times and now seriously exploring a second home / relocation.
```

This is a *plausibility move*, not a full fabrication — plenty of real buyers are in this spot.

---

### 8. They send a pushy follow-up ("¿cuándo podemos vernos?", "le llamo en 5 min")

**Hard close immediately.**

**ES:**
```
Aprecio el seguimiento, pero prefiero ir a mi ritmo. Cuando algo me calce le escribo. Gracias.
```

**EN:**
```
Appreciate the follow-up, but I prefer to move at my own pace. I'll reach out when something fits. Thanks.
```

Don't reply after that. If they push again, ignore.

---

### 9. They don't reply at all (within 72h)

**The silence IS the data.** No close needed. Mark tracker: *"No reply 72h+"*.

## 🛑 CLOSE-LOOP (use when done with any agency — usually after 1-2 exchanges)

**Default close (ES):**
```
Perfecto, muchas gracias. Sigo viendo opciones, le escribo si algo me calza.
```

**Default close (EN):**
```
Perfect, thanks. I'm still looking at options — I'll reach out if something fits.
```

**For Results Realty (after their autoreply + real agent reply):**
```
Gracias, todavía en fase de exploración. Le escribo si algo me calza.
```

## ⚠️ When to bail completely

- **They recognize you / your company.** If they say "¿usted es de Trochai?" or mention having seen your LinkedIn — acknowledge honestly: *"Sí, le estoy midiendo tiempos de respuesta para un estudio. Agradezco el tiempo y no le robo más. Si le interesa lo que hago, le comparto un Loom más tarde."* Then stop.
- **You recognize them / have real history with them.** Skip the shop — too awkward. Move to the next agency.
- **They ask for payment / deposit before sending info.** That's THEIR red flag, not yours — politely disengage.

**Normal flow: 2-3 total messages from you per agency.**
1. Initial mystery shop message (from BATCH-01-QUICK-SEND.md)
2. ONE mid-conversation reply (scenario 2, 3, 4, or 5 above)
3. Close-loop message

**Total time per reply: under 30 seconds.** Don't overthink — copy, paste, close. The goal is data, not relationship.
