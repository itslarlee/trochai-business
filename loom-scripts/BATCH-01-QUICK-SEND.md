## How to use

For each agency:
1. **Save the contact** on your phone with the exact name shown (prefix `Test 01 -` makes cleanup easy — you can bulk-delete later by searching "Test 01")
2. **Copy the message** (tap & hold the code block on mobile, then copy)
3. **Open WhatsApp** → start a new chat with the saved contact → paste → send
4. **Note the send time** on the checkbox
5. **Move to next**
