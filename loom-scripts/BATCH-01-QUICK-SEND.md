# Batch 01 — Quick-Send Checklist (Single Session)

**All 20 in one sitting, 5 min apart.** Should take ~100 minutes.

## How to use

For each agency:
1. **Save the contact** on your phone with the exact name shown (prefix `Test 01 -` makes cleanup easy — bulk-delete later by searching "Test 01")
2. **Copy the message** (tap & hold the code block on mobile)
3. **Open WhatsApp** → start a new chat with the saved contact → paste → send
4. **Note the send time** on the checkbox
5. **Move to next 5 min later**

**⚠️ If someone replies while you're still sending others:** keep `REPLY-PLAYBOOK.md` open in the other tab. Look up the scenario, paste the response, then continue the batch. Don't stop.

**Best window:** 9:00-10:40pm any weeknight. International buyer browsing time. Saturday 2-4pm also works if weeknight doesn't.
