Business strategy, marketing, and GTM documents for the Trochai platform.

## What This Repo Contains

This repo holds all non-code business documents:

| File | Purpose |
|---|---|
| `MARKETING.md` | GTM strategy — ICP, Manifesto, Broadway Show framework |
| `MARKETING-TODO.md` | Action items and progress tracking for marketing execution |
| `PROSPECT-LIST.md` | Target agency list with contacts, discovery call script |

| `VIDEO-SCRIPTS.md` | Short-form video scripts for social media |

## Key Context

- **Product:** Trochai Inbox — AI-powered WhatsApp inbox for real estate agencies in Costa Rica
- **Stage:** Pre-revenue, validating ICP through discovery calls
- **Primary ICP:** Real estate agency owners/managers in Costa Rica (3-20 agents, GAM area)
- **Language:** All outreach in Spanish (ustedeo), English for expat-facing agencies

## Conventions

- All docs are Markdown
- Keep docs as living documents — update as we learn from discovery calls and market feedback
- Outreach tone: respectful but not sycophantic, show research, don't pitch — learn
- Use ustedeo in Spanish (usted, not tú/vos)

## Contributing & Deployment

- Push directly to `main`
- Conventional Commits: `docs:`, `chore:`, etc.
- No CI/CD — this is a docs-only repo
