## 3. Three Short-Form Video Script Concepts

All videos are 9:16 (1080x1920), 30fps, Spanish voiceover (Nathalia voice). Follow the Hook -> Problem -> Solution -> CTA format from the existing video pipeline.

### Script A: "POV: Buscando el mensaje perfecto" (15 seconds)

**Format:** Screen recording style — someone scrolling through WhatsApp, typing, deleting, typing again. Then cuts to the tool.

| Scene | Time | Visual | Voiceover |
|-------|------|--------|-----------|
| 1 (Hook) | 0-3s | POV: WhatsApp chat open, cursor blinking in empty message field. Text overlay: "POV: te escribio un lead nuevo" | "Le escribio un lead. Y usted no sabe que responder." |
| 2 (Problem) | 3-7s | Fast montage: typing "Hola buenas", deleting, typing "Buenos dias, gracias por...", deleting, typing "Hola! Con gusto le...", deleting again | "Hola buenas... no. Buenos dias gracias por... no. Lleva cinco minutos y no ha enviado nada." |
| 3 (Solution) | 7-12s | Cut to Trochai template library. Quick scroll through categories. Click on "Respuesta rapida". Fill in name "Maria". WhatsApp bubble preview appears. One-click copy. | "Con estas plantillas, en diez segundos tiene el mensaje perfecto. Editable. Listo para copiar." |
| 4 (CTA) | 12-15s | Trochai logo + URL: trochai.com/herramientas/plantillas-whatsapp. Text: "11 plantillas gratis" | "Once plantillas gratis. Link en bio." |

**SFX:** Keyboard typing sound (scenes 1-2), notification sound (scene 3 when bubble appears), low kick on CTA.

**Format:** Fast-paced category reveal — each category flies in with a sample message preview.

| Scene | Time | Visual | Voiceover |

Las siete etapas del proceso de venta." |

| 4 (CTA) | 17-20s | Trochai logo + URL. Badge: "100% gratis. Sin registro." | "Gratis. Sin registro. Link en bio." |

**SFX:** Whoosh on each category badge, notification sound on copy click, kick on CTA.

### SEO Actions

1. **Page title and meta description** — Ensure the page's `<title>` and `og:description` include "plantillas WhatsApp inmobiliarias" (ES) and "WhatsApp templates real estate" (EN). Check the page.tsx metadata.

2. **H1 tag** — The hero title should contain the primary keyword naturally. Currently it's defined in the translation files — verify it includes "plantillas de WhatsApp."

3. **Internal links** — Add links to the template library from:
   - The blog post "Como automatizar WhatsApp para inmobiliarias" (high-authority related content)
   - The blog post "WhatsApp Business API para inmobiliarias"
   - The blog post "Mejor CRM para inmobiliarias en Costa Rica"
   - The Products/Inbox page (as a "free tool" callout)

4. **Schema markup** — Consider adding FAQ schema (`FAQPage`) for the 3 FAQ questions already on the page. This can generate rich snippets in Google.

5. **Blog post companion** — Write a dedicated blog post: "11 Plantillas de WhatsApp para Agentes Inmobiliarios [2026]" that embeds/links to the interactive tool. This creates a content cluster: blog post ranks for informational queries, tool page ranks for transactional queries, and they reinforce each other.

6. **Link building** — Every time the tool is shared in FB groups, forums, or direct outreach, it's a potential backlink. The IndieHackers/LinkedIn posts also generate high-authority backlinks to the tool page.
