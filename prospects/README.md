# Prospects — 4-Tier Model

Each file serves a distinct purpose. Not duplicates — complements.

| File | Scope | Use for |
|---|---|---|
| `PROSPECT-LIST.md` | 68 hand-picked agencies, research-heavy | Direct outreach — these are ready to approach |
| `PROSPECT-LIST-EXPANDED.md` | Original list + detailed outreach messages | Reference for message tone + Spanish examples |
| `PROSPECT-LIST-BATCH-3.md` | 336 web-researched leads (MLS, Encuentra24, LinkedIn, directories) | Bulk pool for future batches — less vetted |
| `linkedin-networking-targets.md` | 25+ individual brokers + 3 developers | **NOT for cold outreach.** Strategic LinkedIn targets — comment/like/connect first, then warm into DM |

**Master CSV (not here — see `../leads/master-leads.csv`):** 245 cleaned + prioritized leads with P1/P2/P3 tier + contact info + status tracking.

## Workflow

1. **Mystery shop** from P1 in `leads/master-leads.csv` → see `../loom-scripts/BATCH-01-*.md`
2. **LinkedIn warming** via `linkedin-networking-targets.md` — comment on posts, then connect, then DM
3. **When P1 is exhausted**, pull from Tier 2 (`PROSPECT-LIST-EXPANDED.md`) or Tier 3 (`PROSPECT-LIST-BATCH-3.md`)
